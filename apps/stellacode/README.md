# Stellacode

Stellacode is a local Electron client for Stellaclaw Web channels.

## Run

```bash
cd apps/stellacode
npm install
npm start
```

Add a server in Settings. For a local Stellaclaw Web channel, use the channel base URL such as `http://127.0.0.1:3111` and the token configured by `STELLACLAW_WEB_TOKEN`.
